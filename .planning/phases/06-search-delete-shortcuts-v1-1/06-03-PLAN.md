- Define GlobalShortcutsConfig interface:
  ```ts
  interface GlobalShortcutsConfig {
    onSearch: () => void;
    onNewChat: () => void;
    onEscape?: () => void;
    searchOpen?: boolean;
    dialogOpen?: boolean;
    sheetOpen?: boolean;
  }
  ```
- Export useGlobalShortcuts hook:
  ```ts
  export function useGlobalShortcuts(config: GlobalShortcutsConfig): void
  ```
- Use useHotkeys for each shortcut:
  - 'meta+k' / 'ctrl+k': calls onSearch (preventDefault: true to override browser)
  - 'meta+n' / 'ctrl+n': calls onNewChat (preventDefault: true)
  - 'escape': calls onEscape or closes open overlays (conditional logic)
- Use preventDefault option to override browser defaults for Cmd+K and Cmd+N
- Register shortcuts at document level (scope: undefined for global)

Do NOT:
- Register shortcuts multiple times (react-hotkeys-hook handles deduplication)
- Block shortcuts when typing in input fields (use enableOnFormTags option)
